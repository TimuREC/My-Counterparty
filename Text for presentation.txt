-> Существующая проблема
Я несколько лет работал в налоговой службе и заметил, что одной из частых проблем, с которой сталкиваются предприниматели, - это необходимость проявления должной осмотрительности при выборе контрагентов. Пропуск этого этапа при осуществлении деятельности может обернуться для организации повышенным вниманием со стороны налоговых органов, а также финансовыми потерями, связанными с неисполнением контрагентом своих договорных обязательств.

-> Решение, сервис ФНС
Для того, чтобы облегчить бизнесу решение этой проблемы, в декабре 2018г. Федеральная налоговая служба запустила онлайн-сервис «Прозрачный бизнес». Необходимость внедрения данного сервиса обусловлена ростом числа фирм-однодневок, а также операций с сомнительными организациями. Данный сервис позволяет получить много различных данных из различных официальных источников, с его помощью владельцы компаний могут получить исчерпывающие сведения о своих контрагентах. Однако, на сегодняшний день он все еще работает в тестовом режиме и мобильного приложения для удобного доступа к нему нет.

-> Удобное приложение для использования
Однако, для удовлетворения потребности предпринимателей в первичной проверке своих контрагентов, весь функционал сервиса "Прозрачный бизнес" избыточен. Поэтому в приложении "Мой Контрагент", проверка организаций происходит только по основным факторам риска, позволяющим сформировать общее представление о степени надежности предполагаемого контрагента. Такими факторами я выделил:
- недостоверность адреса регистрации организации
- недостоверность сведений о руководителе и учредителе
- наличие задолженности перед бюджетом
- нахождение организации в процессе ликвидации

Преимуществами использования приложения "Мой Контрагент" являются: экономия времени при проверке контрагентов, а также достоверность данных, поскольку информация предоставляется из официального источника.

-> Кейсы использования
Работа с приложением начинается с экрана поиска.
Пользователь водит ИНН интересующей организации, результат поиска отображается в табличном представлении, по нажатию на организацию происходит переход на экран, где в текстовом виде отображена информация об интересующей пользователя организации. Отсюда же имеется возможность скачать информацию об организации в формате pdf либо поделиться ей.
Также имеется возможность добавлять организации в "Избранное", в таком случае информация будет сохранена на мобильном устройстве, что позволит просматривать ее без доступа к интернету.

-> Использованные технологии
Приложение написано на языке программирования Swift.
С использованием фреймворков UIKit (для создания графического интерфейса) и Core Data (для хранения данных на устройстве).
При разработке я придерживался паттерна проектирования MVC, рекомендуемым компанией Apple.
